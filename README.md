# C
C programming language

1. Setup for VSCode in Windows 
- Download a compiler from https://installc.org/ or from  https://www.codeblocks.org/downloads/binaries/ -> codeblocks-25.03mingw-setup.exe
- Install the compiler and add the path to Windows (if you're using Windows):  
Environmental variables -> path -> New -> path to the MinGW\bin where the compiler is stored
- Create a dir named .vscode in the project main dir  
- Create a file tasks.json  
- Create main.c
- Ctrl + Shift + B - build the main.exe every time before we run the app
- Run the app from the terminal: In the dir where is main.exe type .\main.exe
